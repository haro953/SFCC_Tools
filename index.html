Welcome to SFCC Tools

SFCC Tools is a collection of utilities for working with Salesforce Commerce Cloud (SFCC) projects. Each tool is designed to streamline common development tasks, such as merging XML files, analyzing code, and managing project assets. The goal is to save developers time and reduce errors by providing easy-to-use interfaces for complex operations. Whether you're a seasoned SFCC developer or just getting started, these tools can help you work more efficiently and effectively.

Common Problems, Simple Solutions

Every day developers spend extra time managing data and sanity checking their work. SFCC Tools helps solve these problems by providing easy-to-use tools that simplify common tasks and reduce manual effort.

No Paywalls. No Downloads. No Hassle.

All SFCC Tools are completely free, And no data is collected or stored on any server. Your files are processed entirely in your browser.

Explore The Toolbox

File Scanner Tool

The File scanner tool allows you to upload a file type of your choice and search that file for general or exact matches of text and characters. If found the tool will let you know how many matches were found and highlight those matches in red.

This tool is specifically designed for scanning large files and confirming if your code or term is present without having to open the file in an editor and scroll through it.

File Diff Tool

The file diff tool allows you to quickly upload two files of your choosing and compare those files for differences.

In the future this tool will also have a deep scan option that will compare the files line by line and let you know if a match is not found anywhere in the other file regardless of its position.

File Import Tool

The file import tool allows you to upload two xml files the first file being the file that has newly created metadata and the second file being the file that should receieve that new metadata.

Once uploaded the files display for review and are then able to be merged into a single file containing the metadata from both files. The newly merged file is then displayed and can be copied or downloaded with a timestamp for your convenience.

This tool sloves the age old problem of syncing data as you work on new features for releases without having to worry about losing work or manually copying and pasting metadata between files.
